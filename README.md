# CyberShield AI

CyberShield AI is a modern cybersecurity awareness and protection web app designed to help users understand digital threats, improve online safety, and access quick security guidance through a clean Progressive Web App interface.

## 🚀 Features

- Cybersecurity awareness dashboard
- Modern responsive UI
- Progressive Web App support
- Offline support using service worker
- App install support using manifest file
- Clean and simple user experience
- Security-focused design

## 🛠️ Tech Stack

- HTML
- CSS
- JavaScript
- PWA
- Service Worker
- Web Manifest

📌 How to Run
Download or clone the project.
Open the project folder.
Run index.html in browser.
For best PWA experience, use Live Server in VS Code.
🌐 Live Preview

This project can be deployed on GitHub Pages, Netlify, or Vercel.

📱 PWA Support

CyberShield AI includes:

manifest.json
sw.js
Offline caching support
Installable app structure
🎯 Purpose

The main goal of this project is to create a cybersecurity-themed web application that can educate users about online safety and provide a modern app-like experience.
